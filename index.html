Ledger

Personal finance command center, backed by your own Google Sheet.

```
1
Create a Google Sheet Go to sheets.new and create a blank spreadsheet. Name it anything (e.g. "My Budget").
2
Open Apps Script In your sheet click Extensions → Apps Script. Delete any existing code, paste the script below, then click Save.
3
Deploy as Web App Click Deploy → New deployment. Type: Web app. Execute as: Me. Access: Anyone. Click Deploy, authorize, copy the URL.
4
Paste the URL below Enter your Web App deployment URL and tap Connect.

Your data stays in your own Google account. We never see it.

```
